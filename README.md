📚 Exercícios de Lógica de Programação - Visualg

Este repositório contém exercícios desenvolvidos durante meus estudos de Lógica de Programação utilizando Visualg (Portugol).

Os algoritmos foram criados com o objetivo de praticar conceitos fundamentais da programação e desenvolver raciocínio lógico para resolução de problemas.

📂 Estrutura do repositório

Os exercícios estão organizados de acordo com os conteúdos estudados no curso.
